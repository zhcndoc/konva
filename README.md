# Konva 网站

Konva 的官方网站。使用 Hexo 框架创建。
你可以在 [konvajs/konvajs.github.io](https://github.com/konvajs/konvajs.github.io) 仓库查看生成的文件。


## 入门

安装依赖：

``` bash
$ git clone https://github.com/konvajs/site.git
$ cd site
$ npm install
$ npm install gulp -g
```

生成：

``` bash
$ gulp generate
```

运行服务器：

``` bash
$ gulp server
```
