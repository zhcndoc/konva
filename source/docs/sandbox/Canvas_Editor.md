## 您是否需要一个用于您的网页应用的设计编辑器？

这是一个使用 Konva 制作的完整画布编辑器演示。该演示是由 https://polotno.dev SDK 制作的。

`Polotno` 是一个非常具有主观性的 JavaScript 框架，构建在 `Konva` 和 `React` 之上。您可以使用它制作徽标、演示文稿和社交媒体设计。[polotno.dev](https://polotno.dev/) 支持网页上所有基本的设计编辑功能：文本、图像、滤镜、选择、变换、导出、撤销/重做等许多功能。

您可以轻松地将 Polotno SDK 集成到您的网页应用中，并制作自己的设计编辑器。

**该演示可能尚未针对小屏幕进行优化，因此请[在新窗口中打开](https://codesandbox.io/embed/github/polotno-project/polotno-site/tree/source/examples/polotno-demo?fontsize=11&hidenavigation=1&theme=dark&view=preview)以获得更好的体验。**
