在本演示中，我们将演示如何将放置在画布外的 DOM 元素拖放到舞台上。

如果您需要在触摸设备上启用 DOM 元素的拖放，您将需要一些额外的步骤。有关更多信息，请参阅 [这里](https://mobiforge.com/design-development/touch-friendly-drag-and-drop)。
