title: 如何使用 Vue 和 Konva 监听画布形状的事件？
layout: vue_page

---

通过 `vue-konva`，您可以轻松监听用户输入事件（`click`、`dblclick`、`mouseover`、`tap`、`dbltap`、`touchstart` 等等）以及拖放事件（`dragstart`、`dragmove`、`dragend`）。

有关事件的完整列表，请查看 [on() 方法文档](/api/Konva.Node.html#on)。
