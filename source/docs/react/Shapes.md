title: 使用 React 绘制画布形状
layout: react_page
---

所有 `react-konva` 组件都对应于同名的 `Konva` 组件。除非另有说明，所有适用于 `Konva` 对象的参数都是对应 `react-konva` 组件的有效属性。

基本形状包括：矩形（Rect）、圆形（Circle）、椭圆形（Ellipse）、线（Line）、图像（Image）、文本（Text）、文本路径（TextPath）、星形（Star）、标签（Label）、SVG 路径（SVG Path）、正多边形（RegularPolygon）。你也可以创建自定义形状。
