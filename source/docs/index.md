## 什么是 Konva？

`Konva` 是一个 HTML5 Canvas JavaScript 框架，通过为桌面和移动应用程序提供画布交互性，扩展了 2D 上下文。

`Konva` 支持高性能动画、过渡、节点嵌套、分层、过滤、缓存、事件处理等功能，适用于桌面和移动应用程序，还有更多功能。

您可以在舞台上绘制图形，为它们添加事件监听器，移动它们，缩放它们，并独立于其他形状旋转它们，以支持高性能动画，即使您的应用程序使用成千上万的形状。

该项目始于 [KineticJS](https://github.com/ericdrowell/KineticJS) 的 GitHub 分支。

## 安装 Konva

如果您使用包管理器：

```bash
npm install konva
```

或者直接使用脚本标签：

```html
<script src="https://unpkg.com/konva@9/konva.min.js"></script>
```

或者从 CDN 下载：

- [完整版 konva.js](https://unpkg.com/konva@9/konva.js)
- [压缩版 konva.min.js](https://unpkg.com/konva@9/konva.min.js)
