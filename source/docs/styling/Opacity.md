title: HTML5 Canvas 不透明度教程
---

要设置 Konva 中形状的透明度，我们可以在实例化节点时设置 `opacity` 属性，或者我们可以使用 `opacity()` 方法。

形状的透明度值可以在 0 到 1 之间，其中 0 为完全透明，1 为完全不透明。除非另有说明，所有形状的默认透明度值为 1。

如果你想对多个形状应用透明度而不出现重叠区域，请查看 [透明组演示](/docs/sandbox/Transparent_Group.html)

说明：将鼠标悬停在五边形上以改变其透明度。
