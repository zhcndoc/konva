请阅读完整博客文章以了解制作原型的过程。

该部件是使用 React、Konva 和 react-konva 库制作的。

并不要求此类应用程序必须使用 React。不过在这里使用它是因为主要应用程序（系统所用的地方）是用 React 制作的。
