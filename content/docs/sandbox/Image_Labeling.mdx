图像标注工具用于机器学习中的训练数据标注、审查医学图像以及为反馈标记照片。Konva.js 使得构建这些工具变得容易，因为它具有内置的拖放、形状绘制和画布事件处理功能。

此演示展示了一个使用 React 和 react-konva 构建的可运行的图像标注系统。阅读关于开发过程的 [完整案例研究](https://lavrton.com/case-study-image-labeling-for-machine-learning/)。

该部件是使用 `React`、`Konva` 和 `react-konva` 库制作的。

并不要求此类应用程序必须使用 `React`。不过在这里使用它是因为主要应用程序（系统所用的地方）是用 `React` 制作的。
