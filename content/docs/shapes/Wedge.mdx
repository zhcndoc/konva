要使用 `Konva` 创建一个楔形（饼图片段）形状，可以实例化一个 `Konva.Wedge()` 对象。

有关所有属性和方法的完整列表，请参阅 [楔形 API 参考](/api/Konva.Wedge.html)。

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

<Tabs>
  <TabItem value="Vanilla" default>
```js live vanilla
import Konva from 'konva';

const stage = new Konva.Stage({
  container: 'container',
  width: window.innerWidth,
  height: window.innerHeight
});

const layer = new Konva.Layer();
stage.add(layer);

const wedge = new Konva.Wedge({
  x: stage.width() / 2,
  y: stage.height() / 2,
  radius: 70,
  angle: 60,
  fill: 'red',
  stroke: 'black',
  strokeWidth: 4,
  rotation: -120
});

layer.add(wedge);
```
  </TabItem>
  <TabItem value="React">
```js live react
import { Stage, Layer, Wedge } from 'react-konva';

const App = () => {
  return (
    <Stage width={window.innerWidth} height={window.innerHeight}>
      <Layer>
        <Wedge
          x={window.innerWidth / 2}
          y={window.innerHeight / 2}
          radius={70}
          angle={60}
          fill="red"
          stroke="black"
          strokeWidth={4}
          rotation={-120}
        />
      </Layer>
    </Stage>
  );
};

export default App;
```
